# Cyber Recon Tool

## Overview
The **Cyber Recon Tool** is a cybersecurity dashboard designed for reconnaissance tasks. It includes a port scanner, password checker, IP lookup, and hash generator, all in one interface.  

---

## Features
- **Port Scanner** – Scan open ports on a target host.  
- **Password Checker** – Test password strength.  
- **IP Lookup** – Get info about an IP address.  
- **File/Text Hash Generator** – Generate MD5, SHA-1, or SHA-256 hashes.  
- **User-Friendly Dashboard** – Simple interface for all tools.
